Violano Virtuoso, Audio, Video and eRoll Archive

A Selection of Violano Virtuoso Music - updated October 2024.

A Violano Virtuoso is a mechanical musical instrument comprising a 44 note piano and either one or two violins.
The music for these was originally supplied on paper rolls, much like the Wurlitzer organs.

Music for Violano's which have been converted to midi

Wurlitzer 125 rolls in midi format are free and can be downloaded from the internet. These can be filtered to play on a real Mills Violano Virtuoso which has been adapted to play midi files or to play on a standard midi player.
There is no recognised standard midi file format for Violano's.

For those who wish to make their own midi music for Violano the following midi file format is recommended.
This midi format is already in use for some midified Violano's.

Type 0 midi file with channels as follows:
D string uses midi channel 2
E string uses midi channel 3
G and A strings use midi channel 4
piano uses midi channel 5
the controls (in e-roll format) use midi channel 16

Version 6.0.2 of eplayOrgan has a virtual Violano which can play proper Violano e-rolls, Wurlitzer 125 e-rolls and midi files with the format above.

For midi files the file extensions determine the format played by eplayOrgan's virtual violano:
.mid, normal midi file
.mvv, Violano e-roll
.wvv, Wurlitzer 125 e-roll.

If you play an e-roll using eplayOrgan and record it whilst playing you will get a midi file with the above format. This file can then be played on your Violano. Note that the D and E strings may sometimes try to play more than one note - if this is a problem then the unwanted notes can be removed by editing the midi file. Note also that the controls have already been applied to the midi in this file.
