Miditzer Archive (October 2019)

This is an archive of all the Miditzer Organ Midi Files contained in the Miditzer Forum.
These midi files contain all of their original controls.

All of the music below can be played by eplayOrgan.

eplayOrgan uses a stereo version of the same sound font used by the Miditzers. You can also use eplayOrgan to Translate these files to play correctly on other organs.
