Book Music SA (Special Archive) File Format, February 2025

This is a midi file archive format devised by Craig Williams which encompasses the functions and features of all organs which use Book Music. It is a format which can be directly played by a suitable ePlayOrgan and also translated back to the tracker bar format of the original organ.

Important Notes:
1. Middle C, midi note 60 = C5 etc.
2. Stops are midi control changes 0xBn,0xcv where n=channel, c=0x51 (open) or 0x50 (close), v=stop number.
