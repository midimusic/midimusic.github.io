EplayOrgan

Author: David Back, November 2017 to September 2024

EplayOrgan, Multi Purpose Translating Virtual Pipe Organ

64bit Versions 6.0.4 and 32bit Version 4.2.3-2 plus Portable Versions 6.0.1X and Version 3.1.0.0 all available for download

EplayOrgan is a multi purpose sample based virtual pipe organ with microtonal capability. It is intended to be played by organists and is versatile enough to be useful to everybody with an interest in organs. It will translate, play and record organ midi files for most organs in real time. It will load and be playable almost instantly and does not need an expensive computer with huge amounts of memory.

This organ was designed and tested on a 64 bit Windows 10 system but was always intended to be portable and versatile in that it will run on any 32 or 64 bit Windows system from Windows XP to Windows 11. It will also run on any Mac or Linux system under Wine. You can run Windows on any Mac (using Bootcamp) and eplayOrgan runs perfectly. Latency is negligable under Windows.

A "Portable App" version of eplayOrgan is available, eplayOrganPortable. This also runs on any Windows system or any Mac or Linux system under Wine.

Main Features

  1. Easy and quick to install.
  2. All included organs are fully set up, just connect one or more midi keyboards and a touch screen monitor, select your organ and begin playing. The correct midi channels are automatically set up for each organ.
  3. Optionally version 5.0.0 and onwards can also be set up to play from any standard organ console which has a midi output. This makes the playing options more versatile and is similar to other popular organs. However, as with the other organs, each organ you wish to play has to be individually set up.
  4. The keys, stops, pistons and two swell indicators are all fully animated and their state is easily visible.
  5. Optional Second Touch is available on all organs. See the second touch keys move to show exactly what is happening
  6. Optionally velocity sensitive on every rank. Useful for piano ranks.
  7. All stops are coloured and may be "swiped" open or closed, Manuals have a fully chromatic "swipe".
  8. The stops have automatically selected font sizes, varying with display size which makes them easy to read.
  9. Coupled keys move to show their state.
  10. See the Wurlitzer Band Organ keys and stops moving for the first time ever.
  11. Organs included:-
    Allen: Generic Church Organ and a Universal Organ which is used for translating organ midi files.
    Miditzer: 150, 160, 216 and 260 cinema organs
    Wurlitzer: 125, 150 and 165 band organs
    Hauptwerk: St Anne's Mosely, Burea Church, Paramount 310, Armley Schulze Leeds, Willis Salisbury, Cavaille-Coll Caen both nrpn and sysex, Eisenbarth Friesach
    Estey: Church Organ
    Viscount: Regent
    Grand Orgue: Barton3-7, demoV1 and Clavecin-Blanchet_2cl (harpsichord)
    Modartt Organteq
    Mills Violano Virtuoso
    Plus many more organs, see below.
  12. An organ designer is built in so you can easily design your own organs.
  13. A stop number display simplifies the making of new organs.
  14. A tempo display indicates the current tempo when playing midi files exported from MuseScore.
  15. All organs may be microtonally tuned to any temperament using standard midi sysex commands or .TUN text files.
  16. Outputs from any organ can be translated to Universal Organ format and optionally recorded.
  17. Inputs or recordings in Universal Organ format can be translated to any other organ format and optionally recorded.
  18. New organs can be imported and existing organs exported.
  19. Optional individual .sf2 sound font for each division.
  20. Includes my eplayUniversalOrgan.sf2 sound font and Rich Olsen's Band Organ fonts.
  21. Up to four stereo audio outputs with adjustable reverb and unlimited polyphony.
  22. Built in midi player and recorder using the latest technology.
  23. Multiple playlists. Provides continuous playing of multiple midi files. Playing can be controlled remotely. No other VPO has this feature.
  24. Built in two tone metronome.
  25. Directly plays Wurlitzer Band Organ and Estey e-rolls.
  26. Mills Violano directly plays Mills Violano and Wurlitzer 125 e-rolls.
  27. There are 4 Banks of 38 Divisional Pistons.
  28. A piston stepper selects groups of pistons in order. These groups become General Pistons. This makes playing the organ much easier.
  29. All piston settings can be saved to named files and restored at will.
  30. You are permitted to use eplayOrgan on contrebombarde to upload and demonstrate your work.
  31. Tremolo is fully activated and working on all organs. Each division has separate control.
  32. Autobass is optionally available on Great Division and automatically plays pedals. No other VPO has this.
  33. Swell and expression pedal response is tailored to accurately simulate shutters opening and closing on a real organ. Automatically set up for each organ.
  34. Key inputs from manuals or console can be automatically transposed while playing.
  35. LCD display of stepper and playlist positions is supported (Compatible with HW and GO displays).
  36. eplayOrgan can Read, Write and Play Grand Orgue (SMPTE) midi files.

This organ is perfect for translating organ midi files to play properly on other organs, no other organ can do this.
See: Organ Midi File Translation for details. New organs can be imported and existing organs exported.

All Hauptwerk midi file formats are supported. Stops may be selected using midi nrpn or sysex events.
Master couplers may be used. All will be recorded and played, as appropriate, in midi files.

All GrandOrgue organs are supported via loadable settings files. See below. You can download and run GrandOrgue free from github. It will run on Windows, Mac or some Linux systems. There are many free samplesets available, some of these may be converted to .sf2 format (using Polyphone) to play on eplayOrgan64.

Installing and using Portable Apps

The Portable App installer decompresses the eplayOrganPortable files and copies them into a folder of your choosing. To install two copies of eplayOrganPortable simply run the installer again and copy the files into a different folder.
You can install eplayOrganPortable on to a memory stick and you can then plug this stick into any computer and run it from the stick.
When you unplug the stick nothing on the computer will have been changed (unless you have purposely saved some eplayOrgan midi files on that computer).
You can also install eplayOrganPortable into a "cloud" folder and it will run from there.

To install eplayOrganPortable on Mac or Linux computers, first install Wine, then install eplayOrganPortable into a folder on drive Z: (The other drives will be hidden and inconvenient to use.)
Alternatively or additionally, if Wine is installed you can just plug in a memory stick made for Windows, as above and the organ will run when you double click the .exe file.

Note: The Portable Apps Platform works properly only on Windows computers.

Settings files for GrandOrgue

Import these settings files into your GrandOrgue organ to gain support as indicated.

Use eplayOrgan to drive the GO organ or use eplayOrgan to record midi files for playing on GO.
These settings files are easy to make. See: Organ Midi File Translation for details.

If you use eplayOrgan to make midi files to play on GO - you must load one of the settings files below, or a similar file, to tell the GO midi player how to set stops and swell pedals.
Note: The Barton's Snare Drum is much too loud; reduce it's Gain from 22 to 6db for the best results.
